第5章 线性判别函数 Linear Discriminant Functions

统计模式识别方法有以下分类：

  1. 生成模型 Generative Model
    1. 参数方法 Parametric
      1. Gaussian
      2. Dirichlet
      3. Bayesian network
      4. Hidden Markov model
    2. 非参数方法 Non-Parametric
      1. Histogram density
      2. Parzen window
      3. K-nearest neighbor
  2. 判别模型 Discriminative model
    1. Linear Methods
    2. Neural network
    3. Logistic regression
    4. Decision tree
    5. Kernel(SVM)
    6. Boosting

引言

这里主要介绍Linear Methods。不需要估计类条件概率密度函数，直接找到可用于分类的判别函数。

Bayes分类器，已知类先验概率$p(\omega_i)$和类条件密度函数$p(x|\omega_i)$。任务是估计一个决策函数。特点：需要大量的样本，需要知道某些概率及其形式。

能否利用样本直接设计分类器？有：

  • 线性判别函数、支持向量机、Fisher线性判别函数
  • 广义线性判别函数、非线性判别函数、核学习机

问题形式：$n$个$d$维空间中的样本，每个样本的类别标签已知，共有c个不同的类别。给一个新样本$x\in R^d$，判定它属于$\omega_1,\omega_2,…,\omega_c$中的哪一类别。

基于判别函数的分类器，将样本所在的空间分成若干个互相不重叠的区域，每个区域包含属于同一类的样本。表示界面的函数称为判别函数。 是分类器最常用的表述形式。

线性可分的概念： n个d维空间样本点，如果存在一个线性判别函数能将所有样本正确分类，则称这些样本是线性可分的。
